# Data Catalog for Gold Layer

## Overview

The Gold Layer is the business-level data representation, structured to support analytical
and reporting use cases. It consists of aggregated **views** that provide profitability,
sales, returns, and marketing performance metrics.

---

### 1. gold.monthly_profitability

- **Purpose:** Provides monthly profitability metrics to analyze revenue, costs, profit, refunds, and return performance over time.
- **Source:** `silver.orders`
- **Columns:**

| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| month | DATE | The first day of the month used to group order performance by month. |
| total_orders | INT | Total number of orders placed during the month. |
| total_gross_revenue | NUMERIC(10,2) | Total gross revenue generated during the month before discounts and refunds. |
| total_net_revenue | NUMERIC(10,2) | Total net revenue generated during the month after discounts and refunds. |
| total_cost | NUMERIC(10,2) | Total order-related costs during the month. |
| total_profit | NUMERIC(10,2) | Total profit generated during the month. |
| profit_margin_pct | FLOAT | Profit margin percentage, calculated as total profit divided by total net revenue. |
| returned_orders | INT | Number of orders returned during the month. |
| total_refund_amount | NUMERIC(10,2) | Total amount refunded during the month. |
| return_rate_pct | FLOAT | Percentage of orders returned during the month. |

---

### 2. gold.category_profitability

- **Purpose:** Provides category-level profitability metrics to compare revenue, costs, profit, and return performance across product categories.
- **Source:** `silver.orders`
- **Columns:**

| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| primary_category | NVARCHAR(50) | Product category used to group order performance. |
| total_orders | INT | Total number of orders for the category. |
| total_items_ordered | INT | Total number of items ordered within the category. |
| total_gross_revenue | NUMERIC(10,2) | Total gross revenue generated by the category before discounts and refunds. |
| total_net_revenue | NUMERIC(10,2) | Total net revenue generated by the category after discounts and refunds. |
| total_cost | NUMERIC(10,2) | Total order-related costs for the category. |
| total_profit | NUMERIC(10,2) | Total profit generated by the category. |
| average_order_profit | FLOAT | Average profit per order for the category. |
| profit_margin_pct | FLOAT | Profit margin percentage, calculated as total profit divided by total net revenue. |
| return_rate_pct | FLOAT | Percentage of category orders that were returned. |

---

### 3. gold.marketing_performance

- **Purpose:** Provides marketing performance metrics by month and platform to analyze spend efficiency, conversions, and return on ad spend.
- **Source:** `silver.marketing_spend`
- **Columns:**

| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| month_start_date | DATE | The first day of the month used to group marketing performance. |
| platform | NVARCHAR(50) | Marketing platform where spend and campaign activity occurred. |
| total_spend | NUMERIC(10,2) | Total marketing spend for the month and platform. |
| total_impressions | INT | Total number of ad impressions for the month and platform. |
| total_clicks | INT | Total number of ad clicks for the month and platform. |
| total_conversions | INT | Total number of conversions for the month and platform. |
| total_revenue_attributed | NUMERIC(10,2) | Total revenue attributed to the marketing platform. |
| ctr_pct | FLOAT | Click-through rate percentage, calculated as total clicks divided by total impressions. |
| conversion_rate_pct | FLOAT | Conversion rate percentage, calculated as total conversions divided by total clicks. |
| cpc | FLOAT | Cost per click, calculated as total spend divided by total clicks. |
| cpa | FLOAT | Cost per acquisition, calculated as total spend divided by total conversions. |
| roas | FLOAT | Return on ad spend, calculated as total attributed revenue divided by total spend. |
